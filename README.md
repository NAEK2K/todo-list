A simple todo list built with Node, Express, lowdb, jQuery, and Bootstrap!
Has amazing features such as adding tasks, checking off completed tasks, and even clearing off checked tasks!

Instructions:
- run `node server.js`
- go to `localhost:8080`
- enjoy!
